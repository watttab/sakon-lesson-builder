## 🎯 ระบบนี้คืออะไร?

**GAS Online Lesson Builder** คือแพลตฟอร์มสร้างบทเรียนออนไลน์และแบบทดสอบอัตโนมัติ ที่ออกแบบมาให้คุณครูระดับมัธยมศึกษาสามารถ **สร้างห้องเรียนออนไลน์ได้เองภายในไม่กี่นาที** โดยไม่ต้องมีความรู้ด้านการเขียนโปรแกรม

ระบบนี้ทำงานบน **Google Workspace ของท่านเอง** (Google Sheets + Google Apps Script) ร่วมกับ **AI Gemini Flash** ของ Google ในการเจเนอเรตเนื้อหาและข้อสอบ

## 📋 ขั้นตอนการตั้งค่าระบบ (สำหรับคุณครู)

### 📌 ขั้นที่ 1: ทำสำเนา Google Sheet Template

1. เปิดลิงก์ **Google Sheet Template** ที่วิทยากรแจกให้
2. ไปที่เมนู **ไฟล์ (File)** → **ทำสำเนา (Make a copy)**
3. ตั้งชื่อไฟล์ตามต้องการ (เช่น "บทเรียนวิทยาศาสตร์ ม.5") แล้วกด **ตกลง**
4. ไฟล์สำเนาจะถูกเก็บไว้ใน Google Drive ของท่านเอง

---

### 📌 ขั้นที่ 2: ติดตั้ง Apps Script (หลังบ้านของระบบ)

1. เปิดไฟล์ Google Sheet สำเนาที่เพิ่งสร้าง
2. ไปที่เมนู **ส่วนขยาย (Extensions)** → **Apps Script**
3. จะเปิดหน้าต่างใหม่ขึ้นมา (ตัวแก้ไขสคริปต์)
4. **ลบโค้ดเดิมทิ้งทั้งหมด** แล้วนำ **โค้ด `Code.gs`** ที่วิทยากรเตรียมไว้มาวางแทน
5. คลิกที่ไอคอน **"+"** ข้างคำว่า "ไฟล์" (Files) → เลือก **HTML** → ตั้งชื่อว่า **`index`**
6. ลบโค้ด HTML เดิมทิ้ง แล้วนำ **โค้ด `index.html`** ที่วิทยากรเตรียมไว้มาวางแทน
7. กดปุ่ม **บันทึก** (💾) หรือกด `Ctrl + S`

> **📥 ดาวน์โหลดโค้ด:** ไฟล์ทั้ง 2 สามารถดาวน์โหลดได้จากโฟลเดอร์ [`templates/`](../templates/) ในโปรเจกต์นี้
> - [`code.gs`](../templates/code.gs) — โค้ดหลังบ้าน
> - [`index.html`](../templates/index.html) — หน้าเว็บห้องเรียน

---

### 📌 ขั้นที่ 3: Deploy เผยแพร่เว็บแอป

1. ในหน้า Apps Script ให้คลิกปุ่ม **ทำให้ใช้งานได้ (Deploy)** → **การทำให้ใช้งานได้แบบใหม่ (New deployment)**
2. คลิกไอคอนรูปเฟือง ⚙️ ข้างคำว่า "เลือกประเภท" → เลือก **เว็บแอป (Web app)**
3. ตั้งค่าดังนี้:

| การตั้งค่า | เลือก |
|---|---|
| คำอธิบาย | พิมพ์อะไรก็ได้ เช่น "v1" |
| ดำเนินการในฐานะ | **ฉัน (Me)** |
| ผู้มีสิทธิ์เข้าถึง | **ทุกคน (Anyone)** |

4. กดปุ่ม **ทำให้ใช้งานได้ (Deploy)**
5. ระบบจะแสดง **URL ของเว็บแอป** — ให้คัดลอกเก็บไว้

> **💡 เคล็ดลับ:**  
> URL นี้ใช้ได้ 2 อย่าง:
> - **ลิงก์สำหรับนักเรียน** = เปิดในเบราว์เซอร์เพื่อเข้าเรียน
> - **ลิงก์สำหรับเชื่อมต่อ AI** = ใช้วางในหน้า Lesson Builder

---

### 📌 ขั้นที่ 4: สร้างเนื้อหาบทเรียนด้วย AI

1. เปิดเว็บไซต์ **[sakon-builder.vercel.app](https://sakon-builder.vercel.app)**
2. กรอกข้อมูล 2 อย่าง:
   - **Gemini API Key** — ขอฟรีได้ที่ [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
   - **GAS Web App URL** — ลิงก์ที่คัดลอกจากขั้นตอนที่ 3
3. กด **"ทดสอบการเชื่อมต่อ"** เพื่อตรวจสอบว่าลิงก์ถูกต้อง
4. กรอกข้อมูลวิชา:

| ช่องกรอก | ตัวอย่าง |
|---|---|
| รายวิชา / ระดับชั้น | วิทยาศาสตร์ ม.5 |
| หัวข้อบทเรียนหลัก | เซลล์และการแบ่งเซลล์ |
| รายละเอียดเพิ่มเติม | ออร์แกเนลล์ต่างๆ การแบ่งเซลล์แบบไมโทซิสและไมโอซิส |
| จำนวนข้อสอบ | 5-10 ข้อ (แนะนำ) |

5. กดปุ่ม **"เริ่มต้นสร้างด้วย AI"**
6. รอประมาณ 15-30 วินาที ระบบจะสร้างบทเรียน 3 บท + ข้อสอบ แล้วส่งไปยัง Google Sheet ของท่านอัตโนมัติ

---

### 📌 ขั้นที่ 5: ตั้งค่าเกียรติบัตร (ไม่บังคับ)

1. สร้าง **Google Slides** ใหม่ที่ [slides.new](https://slides.new)
2. ใส่ภาพพื้นหลังเกียรติบัตรที่ต้องการ (หรือออกแบบเอง)
3. สร้างกล่องข้อความแล้วพิมพ์ **ตัวแปร** เหล่านี้ลงไปตรงตำแหน่งที่ต้องการ:

| ตัวแปร | ข้อมูลที่จะแทนที่อัตโนมัติ |
|---|---|
| `{{name}}` | ชื่อ-นามสกุลนักเรียน |
| `{{course}}` | ชื่อวิชาและหัวข้อที่เรียน |
| `{{score}}` | คะแนนที่สอบได้ |
| `{{total}}` | คะแนนเต็ม |
| `{{date}}` | วันที่สอบผ่าน |

4. คัดลอก **Slide ID** จาก URL ของสไลด์
   - ตัวอย่าง URL: `https://docs.google.com/presentation/d/`**`1XyZ...ABCDE`**`/edit`
   - Slide ID คือส่วนที่อยู่ระหว่าง `/d/` กับ `/edit`
5. นำ ID ไปวางในชีต **Config** → แถว `TEMPLATE_SLIDE_ID` → คอลัมน์ B (Value)

## 👨‍🎓 ขั้นตอนการเข้าเรียน (สำหรับนักเรียน)

### 1️⃣ เข้าสู่ระบบ
เปิดลิงก์ที่ครูแจกให้ → กรอก **ชื่อ-นามสกุล** และ **เลขที่** → กดเข้าสู่ระบบ

### 2️⃣ ทำแบบทดสอบก่อนเรียน (Pre-test)
ระบบจะแจ้งเตือนให้ทำ Pre-test ก่อน → กดปุ่ม **"เริ่มทำ"** → ทำข้อสอบให้ครบ → กด **"ส่งคำตอบ"**

### 3️⃣ เรียนเนื้อหาบทเรียน
กดปุ่ม **"เริ่มบทเรียน"** → อ่านเนื้อหาทีละบท → กดปุ่ม **"อ่านจบแล้วและบันทึก"** ทุกบท  
> เมื่ออ่านครบทุกบทแล้ว ระบบจะพากลับมาหน้าแรกอัตโนมัติ

### 4️⃣ ทำแบบทดสอบหลังเรียน (Post-test)
ปุ่ม Post-test จะ **ปลดล็อคเมื่ออ่านบทเรียนครบ** → ทำข้อสอบ → ส่งคำตอบ

### 5️⃣ ดาวน์โหลดเกียรติบัตร
กดปุ่ม **"ดาวน์โหลดใบประกาศ"** → ระบบจะสร้างไฟล์ PDF เกียรติบัตรที่มีชื่อและคะแนนของนักเรียนให้อัตโนมัติ

- ✅ ตรวจสอบว่านักเรียนคนไหน **สอบผ่าน / ไม่ผ่าน**

- ✅ กดปุ่ม **"แก้ไขการวิเคราะห์ประเมินผล"** เพื่อเข้าไปแก้ไขข้อมูลใน Google Sheet โดยตรง

## ⚠️ ข้อควรระวัง

> **🔴 สำคัญมาก:**  
> ทุกครั้งที่แก้ไขโค้ดใน Apps Script จะต้องกด **Deploy → Manage deployments → รูปดินสอ ✏️ → เปลี่ยน Version เป็น "New"** ทุกครั้ง มิฉะนั้นการเปลี่ยนแปลงจะไม่มีผลต่อผู้ใช้งาน

> **📝 หมายเหตุ:**
> - Gemini API Key มี **โควตาฟรีต่อวัน** หากใช้หมดให้รอวันถัดไปหรือสมัคร Key ใหม่
> - ข้อมูลทั้งหมดเก็บอยู่ใน Google Drive ของท่าน **ไม่มีข้อมูลถูกส่งไปเก็บที่อื่น**
> - หากพบปัญหาหน้าเว็บไม่อัปเดต ให้กด `Cmd + Shift + R` (Mac) หรือ `Ctrl + F5` (Windows) เพื่อล้างแคชเบราว์เซอร์
