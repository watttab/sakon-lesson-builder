🎯 ระบบนี้คืออะไร?

GAS Online Lesson Builder คือแพลตฟอร์มสร้างบทเรียนออนไลน์และแบบทดสอบอัตโนมัติ ที่ออกแบบมาให้คุณครูระดับมัธยมศึกษาสามารถ สร้างห้องเรียนออนไลน์ได้เองภายในไม่กี่นาที โดยไม่ต้องมีความรู้ด้านการเขียนโปรแกรม

ระบบทำงานบน Google Workspace ของท่านเอง (Google Sheets + Google Apps Script) ร่วมกับ AI Gemini Flash ของ Google ในการเจเนอเรตเนื้อหาและข้อสอบ

✨ จุดเด่นของระบบ 6 ประการ

คุณสมบัติรายละเอียด
🤖 สร้างเนื้อหาด้วย AIเพียงกรอกชื่อวิชาและหัวข้อ ระบบจะสร้างบทเรียน 3 บทพร้อมข้อสอบให้อัตโนมัติ
📊 ฐานข้อมูลบน Google Sheetsข้อมูลทั้งหมดเก็บใน Google Drive ของท่าน ปลอดภัย 100%
📝 ระบบทดสอบก่อน-หลังเรียนวัดผลสัมฤทธิ์ด้วย Pre-test / Post-test พร้อมกราฟเปรียบเทียบ
🎓 ออกเกียรติบัตรอัตโนมัติสร้าง PDF เกียรติบัตรจาก Google Slides ที่ท่านออกแบบเอง
👨‍🏫 แดชบอร์ดครูผู้สอนดูสรุปคะแนนนักเรียนทุกคนแบบเรียลไทม์
💰 ฟรี ไม่มีค่าใช้จ่ายใช้เครื่องมือฟรีของ Google ทั้งหมด ไม่ต้องเสียค่าเซิร์ฟเวอร์

🔄 ภาพรวมการทำงานของระบบ

ขั้นตอนผู้ดำเนินการรายละเอียด
1 ตั้งค่าระบบครูทำสำเนา Google Sheet Template + Deploy Script
2 สร้างเนื้อหาครู + AIกรอกชื่อวิชา → AI เจเนอเรตบทเรียน + ข้อสอบ → ส่งลง Sheet
3 เรียนรู้ + สอบนักเรียนเปิดลิงก์ → ล็อกอิน → สอบก่อนเรียน → อ่าน → สอบหลังเรียน
4 ดูผล + เกียรติบัตรครู + นักเรียนครูดูสรุปคะแนน / นักเรียนดาวน์โหลดเกียรติบัตร PDF

📋 ขั้นตอนการตั้งค่าระบบ (สำหรับคุณครู)

1ทำสำเนา Google Sheet Template

  1. เปิดลิงก์ Google Sheet Template ที่วิทยากรแจกให้
  2. ไปที่เมนู ไฟล์ (File) → ทำสำเนา (Make a copy)
  3. ตั้งชื่อไฟล์ตามต้องการ (เช่น "บทเรียนวิทยาศาสตร์ ม.5") แล้วกด ตกลง
  4. ไฟล์สำเนาจะถูกเก็บไว้ใน Google Drive ของท่านเอง

2ติดตั้ง Apps Script (หลังบ้านของระบบ)

  1. เปิดไฟล์ Google Sheet สำเนาที่เพิ่งสร้าง
  2. ไปที่เมนู ส่วนขยาย (Extensions) → Apps Script
  3. จะเปิดหน้าต่างใหม่ขึ้นมา (ตัวแก้ไขสคริปต์)
  4. ลบโค้ดเดิมทิ้งทั้งหมด แล้วนำ โค้ด Code.gs ที่วิทยากรเตรียมไว้มาวางแทน
  5. คลิกที่ไอคอน "+" ข้างคำว่า "ไฟล์" (Files) → เลือก HTML → ตั้งชื่อว่า index
  6. ลบโค้ด HTML เดิมทิ้ง แล้วนำ โค้ด index.html ที่วิทยากรเตรียมไว้มาวางแทน
  7. กดปุ่ม บันทึก (💾) หรือกด Ctrl + S
📥 ดาวน์โหลดโค้ด: ไฟล์ทั้ง 2 สามารถดาวน์โหลดได้จาก GitHub ของโครงการ → โฟลเดอร์ templates/

3Deploy เผยแพร่เว็บแอป

  1. ในหน้า Apps Script ให้คลิกปุ่ม ทำให้ใช้งานได้ (Deploy) → การทำให้ใช้งานได้แบบใหม่ (New deployment)
  2. คลิกไอคอนรูปเฟือง ⚙️ → เลือก เว็บแอป (Web app)
  3. ตั้งค่า:
    การตั้งค่าเลือก
    คำอธิบายพิมพ์อะไรก็ได้ เช่น "v1"
    ดำเนินการในฐานะฉัน (Me)
    ผู้มีสิทธิ์เข้าถึงทุกคน (Anyone)
  4. กดปุ่ม ทำให้ใช้งานได้ (Deploy)
  5. ระบบจะแสดง URL ของเว็บแอป — ให้คัดลอกเก็บไว้
💡 URL นี้ใช้ได้ 2 อย่าง: ลิงก์สำหรับนักเรียน (เปิดเข้าเรียน) และ ลิงก์สำหรับเชื่อมต่อ AI (วางในหน้า Lesson Builder)

4สร้างเนื้อหาบทเรียนด้วย AI

  1. เปิดเว็บไซต์ sakon-builder.vercel.app
  2. กรอก Gemini API Key — ขอฟรีได้ที่ aistudio.google.com/apikey
  3. กรอก GAS Web App URL — ลิงก์จากขั้นตอนที่ 3
  4. กด "ทดสอบการเชื่อมต่อ"
  5. กรอกข้อมูลวิชา:
    ช่องกรอกตัวอย่าง
    รายวิชา / ระดับชั้นวิทยาศาสตร์ ม.5
    หัวข้อบทเรียนหลักเซลล์และการแบ่งเซลล์
    รายละเอียดเพิ่มเติมออร์แกเนลล์ต่างๆ ไมโทซิสและไมโอซิส
    จำนวนข้อสอบ5-10 ข้อ
  6. กดปุ่ม "เริ่มต้นสร้างด้วย AI" → รอ 15-30 วินาที → เสร็จ!

👨‍🎓 ขั้นตอนการเข้าเรียน (สำหรับนักเรียน)

  1. เข้าสู่ระบบ — เปิดลิงก์ที่ครูแจก → กรอกชื่อ-นามสกุล + เลขที่ → กดเข้าสู่ระบบ
  2. ทำแบบทดสอบก่อนเรียน (Pre-test) — ระบบจะแจ้งเตือนให้ทำ Pre-test ก่อน → ทำข้อสอบ → ส่งคำตอบ
  3. เรียนเนื้อหาบทเรียน — กดปุ่ม "เริ่มบทเรียน" → อ่านทีละบท → กด "อ่านจบแล้วและบันทึก" ทุกบท
  4. ทำแบบทดสอบหลังเรียน (Post-test) — ปุ่มจะปลดล็อคเมื่ออ่านครบ → ทำข้อสอบ → ส่งคำตอบ
  5. ดาวน์โหลดเกียรติบัตร — กดปุ่ม "ดาวน์โหลดใบประกาศ" → ระบบสร้าง PDF ให้อัตโนมัติ
📌 เมื่อนักเรียนอ่านบทเรียนครบทุกบทแล้ว ระบบจะพากลับมาหน้าแรกอัตโนมัติเพื่อทำแบบทดสอบหลังเรียน

👨‍🏫 แดชบอร์ดครูผู้สอน

เมื่อครูล็อกอินเข้าสู่ระบบ จะสามารถ:

  • ✅ กดปุ่ม "เปิดดูสรุปคะแนนนักเรียน" เพื่อดูตารางคะแนนทุกคน
  • ✅ ตรวจสอบว่านักเรียนคนไหน สอบผ่าน / ไม่ผ่าน
  • ✅ ดูคะแนน Pre-test และ Post-test เปรียบเทียบกัน
  • ✅ กดปุ่ม "แก้ไขการวิเคราะห์ประเมินผล" เพื่อเข้าไปแก้ไขข้อมูลใน Google Sheet โดยตรง

🎓 การตั้งค่าเกียรติบัตร

  1. สร้าง Google Slides ใหม่ที่ slides.new
  2. ใส่ภาพพื้นหลังเกียรติบัตรที่ต้องการ
  3. สร้างกล่องข้อความแล้วพิมพ์ ตัวแปร ลงไป:
ตัวแปรข้อมูลที่จะแทนที่อัตโนมัติ
{{name}}ชื่อ-นามสกุลนักเรียน
{{course}}ชื่อวิชาและหัวข้อที่เรียน
{{score}}คะแนนที่สอบได้
{{total}}คะแนนเต็ม
{{date}}วันที่สอบผ่าน
  1. คัดลอก Slide ID จาก URL (ส่วนระหว่าง /d/ กับ /edit)
  2. นำ ID ไปวางในชีต Config → แถว TEMPLATE_SLIDE_ID → คอลัมน์ B

⚠️ ข้อควรระวัง

🔴 สำคัญมาก: ทุกครั้งที่แก้ไขโค้ดใน Apps Script จะต้องกด Deploy → Manage deployments → รูปดินสอ ✏️ → เปลี่ยน Version เป็น "New" ทุกครั้ง มิฉะนั้นการเปลี่ยนแปลงจะไม่มีผลต่อผู้ใช้งาน
  • Gemini API Key มี โควตาฟรีต่อวัน หากใช้หมดให้รอวันถัดไปหรือสมัคร Key ใหม่
  • ข้อมูลทั้งหมดเก็บอยู่ใน Google Drive ของท่าน ไม่มีข้อมูลถูกส่งไปเก็บที่อื่น
  • หากพบปัญหาหน้าเว็บไม่อัปเดต ให้กด Cmd + Shift + R (Mac) หรือ Ctrl + F5 (Windows)
